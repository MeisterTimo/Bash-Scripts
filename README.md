# Bash-Script Tutorial

**Als Beispiel nehmen wir ein Script namens:** `test.sh`

## Bash-Script Ausführen
Einfach in den Ordner gehen, wo das Skript liegt,  
und dann Folgendes eingeben: `sudo ./test.sh`

## Script nicht ausführbar?
Falls folgende Meldung auftritt: `bash: ./test.sh: Keine Berechtigung`  
muss das Skript erst ausführbar gemacht werden!

## Skript ausführbar machen
Gib dazu im Terminal folgenden Befehl ein: `sudo chmod +x test.sh`

> WICHTIG: Bash-Skripte nie blind ausführen! (erst prüfen)  
> INFO: Getestet auf Debian, meist auch auf anderen Linux-Distributionen nutzbar.
